# Japanese Learning Cards

A very simple Japanese learning app built in a short timeframe to test how quickly an idea can be turned into something tangible.

The intention was not to build a product.  
The intention was to keep it as simple as possible and use it as a practical way to start learning Japanese.

## Why this project

This project started as an experiment:

- to build something useful in a very short time.
- to keep the UI and flow as simple as possible.
- to create a small learning tool for Japanese vocabulary and sentences.
- to understand how fast an idea can become a working prototype.

## What it does

- simple category-based learning.
- English → Japanese flashcards.
- words and sentence examples.
- mobile-friendly layout.
- runs in the browser and can be hosted on GitHub Pages.

## Important note

This is intentionally a lightweight prototype.  
It is not meant to be a fully polished production app, but rather a simple and fast learning experiment.

## Tech

- HTML
- CSS
- Vanilla JavaScript

## Running it

Open `index.html` in your browser, or deploy it via GitHub Pages.
